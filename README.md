音が空に広がるように、

## 🎵 About

**Sorana（ソラナ）** は、あなたの旅の写真と音を組み合わせて、鮮明な記憶を蘇らせるサービスです。

- 📸 **自動訪問地抽出**：Google Photos から旅の履歴を自動生成
- 🔊 **音風景生成**：その場所の環境音、地域音楽を組み合わせ
- 💭 **マルチセンソリー体験**：視覚と聴覚で記憶を呼び覚ます
- 🌅 **日常への統合**：作業用BGM、睡眠導入、旅の計画に
