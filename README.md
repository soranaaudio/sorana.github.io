# Sorana

> 旅の音が、記憶を呼び覚ます  
> Where sound meets memory

音が空に広がるように、あの場所の音が、いま、ここに。

## About

Soranaは、旅の写真と音を組み合わせて、記憶を鮮明に蘇らせるサービスです。

### Features

- 自動訪問地抽出：Google Photosから旅の履歴を自動生成
- 音風景生成：環境音と地域音楽を組み合わせた音の空間
- マルチセンソリー体験：視覚と聴覚で記憶を呼び覚ます
- 日常への統合：作業用BGM、睡眠導入、旅の計画に
