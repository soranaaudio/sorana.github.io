### About

写真は撮る。でも、見返さない。  
旅行は好き。でも、忘れていく。  
あの時の空気感、もう思い出せない？

Soranaは、旅の写真と位置情報から音風景を生成し、視覚と聴覚で記憶を呼び覚ますサービスです。

---

### Features

- 自動訪問地抽出 — Google Photosから旅の履歴を自動生成
- 音風景生成 — 環境音と地域音楽を組み合わせた音の空間
- マルチセンソリー体験 — 視覚と聴覚で記憶を呼び覚ます
